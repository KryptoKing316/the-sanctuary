# Contributing to The Sanctuary

> "Build as unto the Lord. Ship with reverence."

Before contributing, read `CLAUDE.md` — it is the covenant that governs this project.

## Quick Start

```bash
git clone https://github.com/[your-org]/the-sanctuary.git
cd the-sanctuary
npm install
npm run dev
```

## Contribution Tiers

| Tier | Examples | Process |
|------|----------|---------|
| 🟢 Tier 1 — Bug Fixes | Broken buttons, typos, layout bugs | PR → 1 maintainer approval |
| 🟡 Tier 2 — Features | New screens, UI components, prayer categories | Issue first → 2 approvals |
| 🔴 Tier 3 — Theology & Liturgy | Prayer wording, Scripture, communion flow | Eric personally reviews |

## The Question to Ask

Before every PR: **"Does this serve the widow, the orphan, the prisoner?"**

If it doesn't, question whether it belongs.
