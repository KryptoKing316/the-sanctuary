# ✝️ The Sanctuary — Global Prayer & Intercession App

> "Again, truly I tell you that if two of you on earth agree about anything they ask for,
> it will be done for them by my Father in heaven." — Matthew 18:19

**The Sanctuary** is an open-source, decentralized prayer platform that gathers the Body
of Christ in real-time intercession for the least of these — victims of war, trafficking
survivors, widows, orphans, the persecuted church, and the sick.

This is not a wellness app. This is a digital sacred space.

## 🏛️ Core Features

- **Global Prayer Wall** — Live, decentralized intercession feed. Anyone can post.
  Everyone can pray. A counter tracks every prayer offered.
- **The Table** — Scheduled communal communion events. Join thousands worldwide
  at the same moment, guided by liturgy.
- **Prayer Stations** — Immersive intercession for specific missions: war victims,
  trafficking survivors, widows & orphans, the persecuted church.
- **Daily Office** — Morning Lauds, Evening Vespers, Night Compline. Personal
  prayer streaks and journaling.
- **The Rose Window** — Community dashboard. A living rose window SVG that fills
  as prayers are added. Your contribution visualized as part of the whole.

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite) + TypeScript + TailwindCSS + Framer Motion
- **Backend:** Supabase (Auth, Real-time DB, Storage)
- **Fonts:** Cinzel Decorative, IM Fell English, Cinzel
- **Target:** Mobile-first PWA, offline-capable

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/[your-org]/the-sanctuary.git
cd the-sanctuary

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your Supabase URL and anon key

# Start dev server
npm run dev
```

## 🤝 Contributing

We welcome builders, designers, theologians, and intercessors.

### Before You Contribute

1. Read `CLAUDE.md` — it is the covenant that governs this project
2. Read the Tier system to understand what level of review your change needs
3. Search existing Issues before opening a new one

### Contribution Tiers

**🟢 Tier 1 — Bug Fixes**
Broken buttons, typos, layout bugs, failing tests.
Open a PR. One maintainer approval to merge.

**🟡 Tier 2 — Feature Additions**
New UI, new screens, new prayer categories, performance work.
Open an Issue first. Discuss with maintainers. Two approvals to merge.

**🔴 Tier 3 — Theology, Liturgy & Content**
Prayer wording, Scripture, communion flow, sacramental language.
Maintainer-only. Eric personally reviews all Tier 3 changes.

### The Contributor Pledge

By submitting a pull request to this repository, you agree:

> "I am building this as unto the Lord, not merely for users.
> I will flag content concerns before shipping.
> I will not introduce features that profit from spiritual manipulation.
> I welcome correction from the maintainer team.
> I understand that some doors in this house are guarded, and I will
> knock before entering."
